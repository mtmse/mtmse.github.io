OBFL specification

Open Braille Formatting Language is inspired by XSL-FO, where the formatting is described using a specific XML-format with the sole purpose of describing a text's intended rendering.

PEF specification

The Portable Embosser Format (PEF) is a data format for representing braille books, accurately and unambiguously.

braille-utils.impl

Provides embosser, table and paper implementations for the purpose of communicating with embossers.

braille-utils.pef-tools

This contains pef related tools, based on the braille utils api.

dotify.api

Provides the Dotify APIs.

dotify.common

Contains general classes that had to be implemented in order to support the other projects.

dotify.formatter.impl

dotify.formatter.impl contains an implementation of the formatter interfaces of dotify.api.

dotify.hyphenator.impl

Provides hyphenators for many languages. This implementation is based on Franklin Mark Liang's hyphenation algorithm as used in TeX and contains patterns for about 50 languages.

dotify.task.impl

Identify XML formats, validate PEF and OBFL, convert DTBook, epub, html, xml and text to OBFL. Convert OBFL to PEF.

dotify.text.impl

dotify.text.impl contains an implementation of the text interfaces of dotify.api.

dotify.translator.impl

dotify.translator.impl contains an implementation of the translator interfaces of dotify.api.
