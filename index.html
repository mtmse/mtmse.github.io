OBFL specification

Open Braille Formatting Language is inspired by XSL-FO, where the formatting is described using a specific XML-format with the sole purpose of describing a text's intended rendering.

PEF specification

The Portable Embosser Format (PEF) is a data format for representing braille books, accurately and unambiguously.

dotify.library

Dotify library implements functionallity to translate between OBFL and PEF format as well as embosse PEF files
